Release Notes for TExaS

Starter projects for Labs 1, 2, 3, 4, and 5 

UT.12.01x Real-Time Bluetooth Networks - Shape The World

This version fixes the MSP432 bug that incorrectly set the clock speed to 48 MHz. Test EXecute, And Simulate, called TExaS, is a collection of DLL plugins that transform Keil uVision into learning environment for embedded systems. These plugins provide simulation of the Texas Instruments Tiva LaunchPad. The suite also includes a stand-alone application called TExaSdisplay, which provides a serial port interface. When operating with the code embedded on the microcontroller, TExaSdisplay also provides a oscilloscope and logic analyzer functionality. Furthermore, TExaS can be used for automatic grading of labs on the real board for UT.12.01x.
